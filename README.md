# ESP32-Room-Analyser
A device which measures room parameters such as Temperature, Humidity,Illuminance and sound level with the help of sensors like DHT11,LDR,KY038 integrated with Bluetooth for live data stream in the Serial Bluetooth Terminal.

## HARDWARE USED
- ESP32 Devkit
- DHT11 Sensor
- KY038 Sensor 
- LDR
- 2420 Ohm resistor 
- 3 LED's
- Two 220 Ohm resistor 
- LCD I2C Display
